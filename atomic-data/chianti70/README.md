## AtomNeb

**Atomic Data for Ionized Nebulae**

The atomic data were generated for use in [proEQUIB](https://github.com/equib/proEQUIB) and [pyEQUIB](https://github.com/equib/pyEQUIB) obtained from the [CHIANTI atomic database](http://www.chiantidatabase.org/) version 7.0:

* **AtomElj.fits** contains *Energy Levels* (Ej).

* **AtomOmij.fits** contains *Collision Strengths* (Î©ij).

* **AtomAij.fits** contains *Transition Probabilities* (Aij).
